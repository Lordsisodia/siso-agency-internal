# Docs Ledger

Append-only registry for “where did we put it?” across `docs/`.

Format (one line per entry):

- `YYYY-MM-DD` — `<path>` — `<short description>` — `<owner>`

Example:

- 2026-01-04 — `docs/.blackbox/` — added SISO Internal blackbox runtime — shaun
- 2026-01-05 — `docs/.blackbox/` — redesigned blackbox to domain-based runs + agent packages — shaun
