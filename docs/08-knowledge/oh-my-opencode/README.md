## What is Oh-My-OpenCode (Omo)?

**Oh-My-OpenCode** = **O**penCode plugin with **Mo**re features

## Key Innovations

### 1. MCP Integration System ⭐⭐⭐⭐⭐

**What it is:** 8+ curated, production-tested MCP servers

**Servers Included:**
- Context7 - Official documentation lookup
- Exa - Real-time web search
- Grep.app - Ultra-fast GitHub code search
- GitHub - Official GitHub integration
- Playwright - Browser automation
- SQLite - Local database queries
- Chroma - Vector search
- Supabase - Production backend

**Why it matters:**
- Agents access real-world documentation
- Instant access to GitHub examples
- Browser automation for testing
- Database operations without setup

### 2. LSP Tools for Agents ⭐⭐⭐⭐⭐

**What it is:** 10 IDE-like navigation tools

**Tools Available:**
- lsp_hover - Type info at cursor
- lsp_goto_definition - Jump to definition
- lsp_find_references - Find all usages
- lsp_document_symbols - File outline
- lsp_workspace_symbols - Symbol search
- lsp_diagnostics - Error checking
- lsp_rename - Safe refactoring
- lsp_code_actions - Quick fixes
- ast_grep_search - Pattern search
- ast_grep_replace - Pattern replacement

**Why it matters:**
- Agents navigate code like developers
- Faster than grep for navigation
- Structural understanding vs text matching
- Safe automated refactoring

### 3. Multi-Model Orchestration ⭐⭐⭐⭐⭐

**What it is:** 7 specialized agents, 7 optimal AI models

**The 7 Agents:**
- Sisyphus (Claude Opus 4.5) - Main orchestrator
- Oracle (GPT-5.2) - Architecture, debugging
- Librarian (Claude Sonnet 4.5) - Research, examples
- Explore (Grok Code) - Fast navigation
- Frontend-UI/UX (Gemini 3 Pro High) - UI/UX development
- Document-Writer (Gemini 3 Flash) - Technical writing
- Multimodal-Looker (Gemini 3 Flash) - Image/PDF analysis

**Why it matters:**
- Right tool for the job
- 68% cost savings
- Better quality through optimal model selection
