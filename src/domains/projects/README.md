# Projects Domain

Project management, wireframes, and app plans.

## Structure

```
projects/
├── components/       # Reusable project components
├── pages/           # Page-level components
├── details/         # Project detail views
├── features/        # Feature management
├── hooks/           # Custom React hooks
├── services/        # API and business logic
├── types/           # TypeScript definitions
├── userflow/        # User flow diagrams
└── wireframes/      # Wireframe components
```

## Key Features

- **Project Management**: Create, view, and manage projects
- **Wireframes**: Visual wireframe editor and navigation
- **Features**: Feature request tracking and management
- **User Flows**: Interactive user flow diagram editor

## Usage

```tsx
import { ProjectCard, TaskDetailsSheet } from '@/domains/projects';
```
