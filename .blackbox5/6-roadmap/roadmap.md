## How to Use

### 1. Submit a Proposal
Create a file in `00-proposed/` using the template from `templates/proposal-template.md`

### 2. Research Phase
Move to `01-research/` when approved. Investigate feasibility and options.

### 3. Design Phase
Move to `02-design/` when research is complete. Create technical design.

### 4. Planning Phase
Move to `03-planned/` when design is approved. Break down into tasks.

### 5. Active Implementation
Move to `04-active/` when ready to start. Track progress.

### 6. Completion
Move to `05-completed/YYYY/MM/` when done. Document lessons learned.
