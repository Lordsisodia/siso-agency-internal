# Users: SISO Internal Personas

## Primary Personas

### 1. Alex - The Life Hacker

**Demographics**: 28 years old, software developer, remote worker

**Goals**:
- Optimize daily routines for maximum productivity
- Track and improve habits systematically
- Balance work, health, and personal projects

**Frustrations**:
- Too many apps to track different aspects of life
- Can't see patterns across different life areas
- Loses motivation to maintain habits

**How SISO Helps**:
- Lifelock domain provides structured daily routines
- XP system provides motivation and progress tracking
- Analytics reveal patterns and insights

**Key SISO Features**:
- Daily checkout and tracking
- XP and achievements
- Task management with deep work sessions
- Analytics and progress visualization

## User Journey Patterns

### New User Onboarding

1. **Discovery**: User learns about SISO through referral or search
2. **Setup**: Create account, complete initial profile
3. **First Domain**: Start with Lifelock for daily routines
4. **Expansion**: Add Tasks domain as needs grow
5. **Integration**: Use multiple domains as lifestyle evolves

### Power User Evolution

1. **Casual User**: Uses 1-2 domains occasionally
2. **Regular User**: Uses 3-4 domains daily
3. **Power User**: Uses all domains, tracks everything, earns XP regularly
