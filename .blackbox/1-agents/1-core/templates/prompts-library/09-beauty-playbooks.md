## CONTEXT

Use the shared Deep Research Context Pack (copy/paste into your run as needed):

## PROMPT

TASK:
Research operational playbooks and standard workflows for beauty DTC brands at ~10k orders/month, especially around:

- Returns/exchanges policies and best practices
- Customer support deflection patterns (“where is my order”, return policy, replacements)
- Inventory/stockout handling and preorder/backorder messaging
- Bundles/kits and seasonal promos
- Review/UGC workflows
- Shipping/fulfillment exceptions

Deliver:
- A list of 25–50 “default workflows” we can ship as templates (“playbooks”)
- For each playbook: trigger, steps, human approval points, KPIs, failure modes

OUTPUT REQUIREMENTS:
- Cite sources (operator blogs, agency playbooks, tool documentation, case studies).
- End with “Top 10 playbooks to implement first” for the current client category.
