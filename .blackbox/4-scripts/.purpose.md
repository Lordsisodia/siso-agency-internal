# .purpose.md

## Directory: 4-scripts/

**Purpose:** All executable scripts, libraries, and utilities for Blackbox4.

**Contains:**
- **Libraries:** `lib/` - Reusable code shared across agents (context variables, hierarchical tasks, spec creation, ralph runtime)
- **Agent Scripts:** `agents/` - Agent execution and handoff scripts
- **Planning Scripts:** `planning/` - Plan creation and management scripts
- **Integration Scripts:** `integration/` - Cross-component integration scripts
- **Validation Scripts:** `validation/` - Validation and verification scripts
- **Questioning:** `questioning/` - Sequential questioning workflow scripts
- **PRD Templates:** `prd-templates/` - PRD generation templates
- **Testing:** `testing/` - Test infrastructure and scripts

**Used By:**
- All agents (for shared functionality)
- Ralph Runtime (for autonomous execution)
- Developers (for running workflows)
- CI/CD (for validation and testing)

**Owner:** Core Team
**Stability:** High (changes require coordination)

**Key Files:**
- `lib/*/` - Library directories for each phase
- `agents/agent-handoff.sh` - Agent handoff system
- `planning/new-plan.sh` - Plan creation
- `testing/phase*/validate-phase*.sh` - Phase validation scripts
