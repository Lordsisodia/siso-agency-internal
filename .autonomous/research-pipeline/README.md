# Research Pipeline Directory

**Purpose:** Store research artifacts, query banks, and investigation outputs.

---

## Directory Structure

```
research-pipeline/
├── queries/          # Saved search queries
```

## Research Workflow

1. Define research question
2. Create query bank
3. Execute searches
4. Document findings
5. Store in `findings/`
